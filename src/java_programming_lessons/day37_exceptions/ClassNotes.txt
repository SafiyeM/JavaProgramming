final vs finalize vs finally:

	final: keyword ==> to indicate that the variable, method, class unchangeable

	finalize: method ===> Garbage Collector use it to collect the objects that are eligible for garbage collection
	                      Every object has 'finalize' method

	finally: block ===> can only be used with try&catch
